# 高阶导数

- ## 常用的高阶导数公式
- $\displaystyle (\sin x)^{(n)} = \sin\left(x + n \cdot \frac{\pi}{2}\right)$
    - 推广：$\displaystyle [\sin(ax+b)]^{(n)} = a^n \sin\left(ax+b + n \cdot \frac{\pi}{2}\right)$
- $\displaystyle (\cos x)^{(n)} = \cos\left(x + n \cdot \frac{\pi}{2}\right)$
    - 推广：$\displaystyle [\cos(ax+b)]^{(n)} = a^n \cos\left(ax+b + n \cdot \frac{\pi}{2}\right)$
- $\displaystyle (u \pm v)^{(n)} = u^{(n)} \pm v^{(n)}$
- $\displaystyle (uv)^{(n)} = \sum_{k=0}^{n} \mathrm{C}_{n}^{k} u^{(k)} v^{(n-k)}$（莱布尼茨公式）
