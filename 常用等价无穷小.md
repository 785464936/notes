# 常用等价无穷小

### 一、按阶数排序

#### 1 阶无穷小（\(\sim kx\)）

- $\sin x \sim x$
- $\tan x \sim x$
- $\arcsin x \sim x$
- $\arctan x \sim x$
- $\ln(1 + x) \sim x$
- $e^x - 1 \sim x$
- $a^x - 1 \sim x \ln a$
- $(1 + x)^\alpha - 1 \sim \alpha x$
- $\log_a(1 + x) \sim \dfrac{x}{\ln a}$

#### 2 阶无穷小（\(\sim kx^2\)）

- $1 - \cos x \sim \dfrac{1}{2}x^2$
- $\ln(\cos x) \sim -\dfrac{1}{2}x^2$
- $e^{x^2} - 1 \sim x^2$
- $\sqrt{1 + x^2} - 1 \sim \dfrac{1}{2}x^2$
- $\sqrt[3]{1 + x^2} - 1 \sim \dfrac{1}{3}x^2$
- $\ln(1 + x^2) \sim x^2$

#### 3 阶无穷小（\(\sim kx^3\)）

- $x - \sin x \sim \dfrac{1}{6}x^3$
- $\tan x - x \sim \dfrac{1}{3}x^3$
- $\arcsin x - x \sim \dfrac{1}{6}x^3$
- $\arctan x - x \sim -\dfrac{1}{3}x^3$
- $\tan x - \sin x \sim \dfrac{1}{2}x^3$
- $\sin x - x \cos x \sim \dfrac{1}{3}x^3$
- $e^{x^3} - 1 \sim x^3$
- $\ln(1 + x^3) \sim x^3$

### 二、按函数类型分组

#### 三角函数类

- $\sin x \sim x$
- $\tan x \sim x$
- $\arcsin x \sim x$
- $\arctan x \sim x$
- $1 - \cos x \sim \dfrac{1}{2}x^2$
- $x - \sin x \sim \dfrac{1}{6}x^3$
- $\tan x - x \sim \dfrac{1}{3}x^3$
- $\arcsin x - x \sim \dfrac{1}{6}x^3$
- $\arctan x - x \sim -\dfrac{1}{3}x^3$
- $\tan x - \sin x \sim \dfrac{1}{2}x^3$

#### 指数对数类

- $e^x - 1 \sim x$
- $a^x - 1 \sim x \ln a$
- $\ln(1 + x) \sim x$
- $\ln(1 + x^2) \sim x^2$
- $\ln(1 + x^3) \sim x^3$

#### 幂函数类（$(1+u)^\alpha - 1$ 型）

- $(1 + u)^\alpha - 1 \sim \alpha u$（通用公式，$u \to 0$）

**常见特例：**

- $\sqrt{1 + x} - 1 \sim \dfrac{1}{2}x$
- $\sqrt[3]{1 + x} - 1 \sim \dfrac{1}{3}x$
- $\sqrt{1 + x^2} - 1 \sim \dfrac{1}{2}x^2$
- $\sqrt[3]{1 + x^2} - 1 \sim \dfrac{1}{3}x^2$

---

### 三、复合函数等价规律

若 $u \sim k x^m$（$u$ 是 $m$ 阶无穷小），则：

- $\ln(1 + u) \sim u \sim k x^m$
- $e^u - 1 \sim u \sim k x^m$
- $a^u - 1 \sim (\ln a) u \sim (\ln a) k x^m$
- $(1 + u)^\alpha - 1 \sim \alpha u \sim \alpha k x^m$

---

### 四、重要提醒

- 加减法慎用等价替换，若主项抵消，需展开到更高阶
- 系数记忆技巧：
  - $x - \sin x \sim \dfrac{1}{6}x^3$
  - $\tan x - x \sim \dfrac{1}{3}x^3$
  - $\arcsin x - x \sim \dfrac{1}{6}x^3$
  - $\arctan x - x \sim -\dfrac{1}{3}x^3$
- $\tan x - \sin x$ 用乘积法推导：$$\tan x - \sin x = \tan x (1 - \cos x) \sim x \cdot \dfrac{1}{2}x^2 = \dfrac{1}{2}x^3$$
- 通用公式 $(1+u)^\alpha - 1$ 可解决大多数根式类问题
