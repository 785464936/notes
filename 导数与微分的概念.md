# 导数与微分的概念

## 导数的概念

常用等价形式$f^{\prime}(x_{0})= \displaystyle \lim_{ \Delta x \to 0 } \dfrac{f(x_{0}+\Delta x)-f(x_{0})}{\delta x} = \lim_{ x \to x_{0} } \dfrac{f(x)-f(x_{0})}{x - x_{0}}=\lim_{ h \to 0 } \dfrac{f(x_{0}+h)-f(x_{0})}{h}$
