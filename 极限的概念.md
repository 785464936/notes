## 数列的极限

定义: $\lim_{n \to \infty}x_n=a:\forall \varepsilon>0,\exists N>0$，当 $n>N$ 时，恒有 $|x_n-a|<\varepsilon.$ 

> [!Tip]
> -  $\varepsilon$ 与 $N$ 的作用  
> - 几何意义  
> - 数列 $\{x_{n}\}$ 的极限与前有限项**无关**  
> - $\lim_{ n \to \infty }=a \Leftrightarrow \lim_{ k \to \infty }x_{2k-1}=\lim_{ k \to \infty }x_{2k}=a$

## 函数的极限

### 自变量趋于无穷大时函数的极限

定义: $\lim_{x \to +\infty}f(x)=A: \forall \varepsilon>0,\exists N>0$，当 $x>X$ 时，恒有 $|f(x)-A|<\varepsilon.$ 

$\lim_{x \to -\infty}f(x)=A: \forall \varepsilon>0,\exists N>0$，当 $x<-X$ 时，恒有 $|f(x)-A|<\varepsilon.$ 

$\lim_{x \to \infty}f(x)=A: \forall \varepsilon>0,\exists N>0$，当 $|x|>X$ 时，恒有 $|f(x)-A|<\varepsilon.$

> [!定理]
> $\lim_{ x \to \infty }f(x)=A\Leftrightarrow\lim_{ x \to -\infty }f(x)=\lim_{ x \to +\infty }f(x)=A$  

### 自变量趋于有限值时函数的极限

定义: $\lim_{ x \to x_{0} }f(x)=A:\forall\varepsilon>0,\exists \delta>0$, 当 $0<|x-x_0|$ 时, 恒有 $|f(x)-A<\varepsilon.$ 

> [!tip]
> - $\varepsilon$ 与 $\delta$ 的作用, $\varepsilon$ 的任意性
> - 几何意义
> - $x\to x_{0}$ 但 $x\neq x_{0}$
> - $\lim_{ x \to x_{0} }$ 与 $f(x_{0})$ 无关

左极限: $\lim_{ x \to x_{0} }f(x)=f(x_{0}^{-})=f(x_{0}-0)$

左极限: $\lim_{ x \to x_{0}^{\prime} }f(x)=f(x_{0}^{+})=f(x_{0}+0)$

> [!定理]
> $\lim_{ x \to x_{0} }f(x)=A\Leftrightarrow\lim_{ x \to x_{0}^{-} }f(x)=\lim_{ x \to x_{0}^{+} } f(x)=A$  

需要分左右极限求极限的问题

- 分段函数在分界点处的极限 (分界点两侧函数表达式不同)
- $e^{\infty}$ 型极限
    - $\lim_{ x \to 0 }e^{\frac{1}{x}},\lim_{ x \to \infty }e^{x},\lim_{ x \to \infty } e^{-x}$
- $\arctan \infty$ 型极限
    - $\lim_{ x \to 0 } \arctan \frac{1}{x},\lim_{ x \to \infty } \arctan x$ 极限不存在
