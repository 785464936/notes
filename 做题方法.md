# 做题方法

- 等式右边任意阶可导 $\Leftrightarrow$ 等式左边任意阶可导

- 介值定理：$f(a)f(b) < 0 \to f(\xi) = 0$
- [[微分中值定理#费马引理|费马引理]]：$f(x_{0})$ 在 $(a,b)$ 取极值，$f^{\prime}(x_{0}) = 0$ ，取极值可以和 $f^{\prime\prime}(x)$ 取得联系
- [[微分中值定理#罗尔定理|罗尔定理]]：$f(a) = f(b), f^{\prime}(\xi) = 0$

- 二阶导为 0 的点不一定为拐点

- $f^{(n)}(x) \neq 0$ （前 n-1 阶全为 0）
    - n 为奇数 -> 拐点
    - n 为偶数 -> 不是拐点
