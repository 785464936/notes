# 常用求极限的 8 种方法

## 利用基本极限求极限

### 常用基本极限

- $\displaystyle\lim_{ x \to 0 } \dfrac{\sin x}{x} =1$
- $\displaystyle\lim_{ x \to 0 }(1+x)^{\frac{1}{x}} = \mathrm{e}$
- $\displaystyle\lim_{ x \to \infty }\left( 1+\dfrac{1}{x} \right)^{x} = \mathrm{e}$
- $\displaystyle \lim_{ x \to 0 } \dfrac{a^{x}-1}{x} = \ln a$
- $\displaystyle \lim_{ n \to \infty } \sqrt[n]{ n } = 1$
- $\displaystyle \lim_{ n \to \infty }\sqrt[n]{ a } = 1(a>0)$
- $$\displaystyle \lim_{ x \to \infty } \dfrac{a_{n}x^{n}+a_{n-1}x^{n-1}+\cdots + a_{1}x+a_{0}}{b_{m}x^{m}+b_{m-1}x^{m-1}+\cdots + b_{1}x+b_{0}} = \begin{cases}
\dfrac{a_{n}}{b_{m}},  & n=m, \\
0,  & n<m, \\
\infty,  & n>m.
\end{cases}$$
- $$
\displaystyle  \lim_{ n \to \infty } x^{n}= \begin{cases}
0, & \lvert x \rvert <1, \\
\infty, &  \lvert x \rvert >1, \\
1, & x = 1 , \\
\text{不存在}， & x = -1.
\end{cases}
$$
- $$
\displaystyle \lim_{ n \to \infty } \mathrm{e}^{nx}= \begin{cases}
0, & x<0, \\
+\infty, & x>0, \\
1, & x = 0.
\end{cases}
$$
### "$1^{\infty}$"型极限常用结论

若 $\lim \alpha(x) = 0, \lim\beta(x) = \infty,\text{且 }\lim\alpha(x)\beta(x) = A$，则
$$
\lim [1+\alpha(x)]^{\beta(x)} = \mathrm{e}^{A}
$$
> [!tip]
> 可以归纳为以下三步  
> 1. 写标准形式：$\text{原式} = \lim [1+\alpha(x)]^{\beta(x)}$
> 2. 求极限：$\lim\alpha(x)\beta(x) = A$
> 3. 写结果：$\text{原式} = \mathrm{e}^{ A }$

## 利用等价无穷小代换求极限

- 代换原则
    - 乘除关系可换：
        - $若\alpha \sim \alpha_{1}, \beta \sim \beta_{1}, 则 \lim \dfrac{\alpha}{\beta} = \lim \dfrac{\alpha_{1}}{\beta} = \lim \dfrac{\alpha}{\beta_{1}} = \lim \dfrac{\alpha_{1}}{\beta_{1}}$
    - 加减关系在一定条件下可换：
        - $若\alpha \sim \alpha_{1}, \beta \sim \beta_{1},且 \lim \dfrac{\alpha_{1}}{\beta_{1}} = A \neq 1, 则 \alpha - \beta \sim \alpha_{1} - \beta_{1}$
        - $若\alpha \sim \alpha_{1}, \beta \sim \beta_{1},且 \lim \dfrac{\alpha_{1}}{\beta_{1}} = A \neq -1, 则 \alpha + \beta \sim \alpha_{1} + \beta_{1}$

## 利用有理运算法则求极限

**有理运算法则** $若\lim f(x) = A, \lim g(x) = B. 那么$ 

- $\lim [f(x)\pm g(x)] = \lim f(x) \pm \lim g(x) = A \pm B$
- $\lim [f(x)g(x)] = \lim f(x) \cdot \lim g(x) = AB$
- $\lim \left[ \dfrac{f(x)}{g(x)} \right] = \dfrac{\lim f(x)}{\lim g(x)} = \dfrac{A}{B}(B \neq 0)$

> [!tip]
> - $存在 \pm 不存在 = 不存在$
> - $不存在 \pm 不存在 = 不存在$
> - $存在 \times(\div) 不存在 = 不一定$
> - $不存在 \times(\div) 不存在 = 不一定$

> [!tip] 常用结论
> - $\lim f(x) = A \neq 0 \implies \lim f(x) g(x) = A \lim g(x)$
>     - 即：先求极限非 0 的因子的极限
> - $\lim \dfrac{f(x)}{g(x)}存在, \lim g(x) = 0 \implies \lim f(x) = 0$
> - $\lim \dfrac{f(x)}{g(x)} = A \neq 0, \lim f(x) = 0 \implies \lim g(x) = 0$

## 利用洛必达法则求极限

[[洛必达法则]]

> [!tip] 适用类型
> 七种不定式：
> $\dfrac{0}{0}$, 
> $\dfrac{\infty}{\infty}$, 
> $\infty - \infty$,
> $0 \cdot \infty$,
> $1^{\infty}$,
> $\infty^{0}$,
> $0 ^{0}$
> > [!tip] 转化规律
> > 前两种直接使用，后五种均可化为前两种
> > $$
> > 
\frac{0}{0},\frac{\infty}{\infty} \Leftarrow
\begin{cases}
0 \cdot \infty \Leftarrow
\begin{cases}
1^\infty \\
\infty^0 \\
0^0
\end{cases} \\
\infty - \infty
\end{cases}
$$

> [!note] 使用洛必达法则应注意的问题
> - 先检验条件是否满足
>     - 如果题目为函数 $n$ 阶可导，只能洛到一阶导函数 $f^{\prime}(x)$
> - 使用后如果问题仍是未定型极限，且符合条件，可以再次使用
> - 如果 $\dfrac{0}{0}$ 型或 $\dfrac{\infty}{\infty}$ 型极限中的函数含有极限**非 0** 的因子，可以单独求极限，不必参与洛必达法则的运算
> - 如果能配合洛必达法则进行等价无穷小量代换或恒等变形，可以简化运算

## 利用夹逼准则求极限

> [!tip] 常用结论
> 求 $\displaystyle \lim_{ n \to \infty }\sqrt[n]{ a_{1}^{n}+ a_{2}^{n} + \cdots + a_{m}^{n} }$ , 其中 $a_{i}>0$  
> $$
> \begin{align}
> \sqrt[n]{a^{n}} \leq \sqrt[n]{ a_{1}^{n}+ a_{2}^{n} + \cdots + a_{m}^{n} } \leq \sqrt[n]{ma^{n}}, \\
> \lim_{ n \to \infty } \sqrt[n]{a^{n}} = a, \lim_{ n \to \infty } \sqrt[n]{m \cdot a^{n}} = a,
> \end{align}
> $$
> $$\boxed{\displaystyle \lim_{ n \to \infty }\sqrt[n]{ a_{1}^{n}+ a_{2}^{n} + \cdots + a_{m}^{n} } = \max \{ a_{i}  \} = a}$$
