# 常用求极限的 8 种方法

## 利用基本极限求极限

### 常用基本极限

- $\displaystyle\lim_{ x \to 0 } \dfrac{\sin x}{x} =1$
- $\displaystyle\lim_{ x \to 0 }(1+x)^{\frac{1}{x}} = \mathrm{e}$
- $\displaystyle\lim_{ x \to \infty }\left( 1+\dfrac{1}{x} \right)^{x} = \mathrm{e}$
- $\displaystyle \lim_{ x \to 0 } \dfrac{a^{x}-1}{x} = \ln a$
- $\displaystyle \lim_{ n \to \infty } \sqrt[n]{ n } = 1$
- $\displaystyle \lim_{ n \to \infty }\sqrt[n]{ a } = 1(a>0)$
- $$\displaystyle \lim_{ x \to \infty } \dfrac{a_{n}x^{n}+a_{n-1}x^{n-1}+\cdots + a_{1}x+a_{0}}{b_{m}x^{m}+b_{m-1}x^{m-1}+\cdots + b_{1}x+b_{0}} = \begin{cases}
\dfrac{a_{n}}{b_{m}},  & n=m, \\
0,  & n<m, \\
\infty,  & n>m.
\end{cases}$$
- $$
\displaystyle  \lim_{ n \to \infty } x^{n}= \begin{cases}
0, & \lvert x \rvert <1, \\
\infty, &  \lvert x \rvert >1, \\
1, & x = 1 , \\
\text{不存在}， & x = -1.
\end{cases}
$$
- $$
\displaystyle \lim_{ n \to \infty } \mathrm{e}^{nx}= \begin{cases}
0, & x<0, \\
+\infty, & x>0, \\
1, & x = 0.
\end{cases}
$$
### "$1^{\infty}$"型极限常用结论

若 $\lim \alpha(x) = 0, \lim\beta(x) = \infty,\text{且 }\lim\alpha(x)\beta(x) = A$，则
$$
\lim [1+\alpha(x)]^{\beta(x)} = \mathrm{e}^{A}
$$
> [!tip]
> 可以归纳为以下三步  
> 1. 写标准形式：$\text{原式} = \lim [1+\alpha(x)]^{\beta(x)}$
> 2. 求极限：$\lim\alpha(x)\beta(x) = A$
> 3. 写结果：$\text{原式} = \mathrm{e}^{ A }$
