# 转置矩阵

$A = (a_{ij})_{m\times n}\quad A^{\mathrm{T}} = (a_{ji})_{n\times m}$

$|A^{\mathrm{T}}| = |A|$

### 运算律

矩阵的转置也是一种运算，满足下面的运算规律（假设运算都是可行的）：

- 反身性 $(A^{\mathrm{T}})^{\mathrm{T}} = A$
- 数乘 $(kA)^{\mathrm{T}} = kA^{\mathrm{T}}$ （$k$ 为任意数）
- 加法法则 $(A+B)^{\mathrm{T}} = A^{\mathrm{T}} + B^{\mathrm{T}}$ #易错 
- 乘法法则 $(AB)^{\mathrm{T}} = B^{\mathrm{T}}A^{\mathrm{T}}$
- $(A_1 A_2 \cdots A_k)^{\mathrm{T}} = A_k^{\mathrm{T}} \cdots A_2^{\mathrm{T}} A_1^{\mathrm{T}}$

## 对称矩阵与反反对称矩阵

对称：$a_{ij} = a_{ji}$，即 $A=A^{\mathrm{T}}$

反对称：$a_{ij} = -a_{ji}$，即 $A=-A^{\mathrm{T}}$

有以下结论：
- 设 $B$ 是一个 $m \times n$ 矩阵，则 $B^{\mathrm{T}}B$ 和 $BB^{\mathrm{T}}$ 都是对称矩阵
- 对反对称阵 $A=(a_{ij})_{n \times n}$ 来说，必有 $a_{ii}=0 \ (i=1,2,\cdots,n)$ 
- 对奇数阶的反对称阵 $A=(a_{ij})_{n \times n}$ 来说，必有 $|A|=0$
- 任意一个 $n$ 阶方阵都可以表示成一个对称矩阵与一个反对称矩阵的和
- 数字的转置等于其本身
