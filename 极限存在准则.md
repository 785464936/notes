# 极限存在准则

## 夹逼准则

若存在 $N$, 当 $n>N$ 时, $x_{n}\leq y_{n}\leq z_{n},\displaystyle\lim_{ n \to \infty }x_{n}=\displaystyle\lim_{ n \to \infty }z_{n} = a$, 则 $\displaystyle\lim_{ n \to \infty }y_{n}=a$. 

> [!tip]
> 多用在数列求 n 项和题目中

> [!e.g.]
> 求极限 $\displaystyle\lim{ n \to \infty }\left[ \frac{n}{n^{2}+1} + \frac{n}{n^{2}+2} + \frac{n}{n^{2}+3} + \cdots + \frac{n}{n^{2}+n} \right]$

## 单调有界准则

单调增, 有上界的数列必有极限

单调减, 有下界的数列必有极限
