# 常见特殊矩阵

### 零矩阵

元素全为 0, 记为 $O$

### 对角矩阵

除主对角线外其余元素均为 0, 记为 $\Lambda$

简记为

$$\Lambda = \mathrm{diag}(a_{11},a_{22}, \cdots, a_{nn})$$

### 数量矩阵

主对角线元素全相等的对角矩阵

$$\Lambda = \mathrm{diag}(k, k, \cdots, k)$$
### 单位矩阵

主对角线元素全为 $1$ 的数量矩阵，记为 $E$ 或 $I$，$E_n$ 或 $I_n$

其作用与数字运算中的 1 类似

### 行阶梯矩阵

全 0 行都位于矩阵的底部

非 0 首元逐行右移

### 行最简矩阵

非零行的非零首元均为 1

非零首元所在列的其他元素均为 0
