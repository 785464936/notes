# 向量

## 向量

$n$ 个数组成的有序数组称为一个 $n$ 维向量，简称向量

> 考研范围内只讨论实向量

### 向量相等

$\alpha$ 和 $\beta$ 对应的分量都相等，即 $\alpha_{i}=\beta_{i}$，称两个向量相等

## 线性运算

加

减

数乘

### n 维向量的线性运算的性质

以下 $\alpha, \beta, \gamma$ 是 $n$ 维向量，$k, l$ 是常数

- 零性质：$0\alpha = 0$
- 对数字的分配律：$(k + l)\alpha = k\alpha + l\alpha$
- 对数字的结合律：$k(l\alpha) = (kl)\alpha$
- 对向量的分配律：$k(\alpha + \beta) = k\alpha + k\beta$
- 向量加法交换律：$\alpha + \beta = \beta + \alpha$
- 向量加法结合律：$(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)$

## 向量内积与向量的模

$\alpha, \beta$ 均为 $n$ 维列向量

### 内积

$$
(\alpha,\beta)=\alpha^{\mathrm{T}}\beta=\sum_{i=1}^{n} a_{i}b_{i}
$$

### 向量内积的性质

其中，$k$ 是数，$\alpha, \beta, \gamma$ 均为 $n$ 维向量

- $(\alpha, \beta) = (\beta, \alpha)$
- $(k\alpha, \beta) = k(\alpha, \beta)$
- $(\alpha + \beta, \gamma) = (\alpha, \gamma) + (\beta, \gamma)$  
> 向量内积的分配律.
- $(\alpha, \alpha) = \alpha^{\mathrm{T}} \alpha \ge 0$ , 且 $(\alpha, \alpha) = \alpha^{\mathrm{T}} \alpha = 0 \Leftrightarrow \alpha = 0$  

> 2.3.条共同组成向量内积的 **双线性性质** :  
> $(k\alpha + l\beta, \gamma) = k(\alpha, \gamma) + l(\beta, \gamma)$ 或  
> $(\alpha, k\beta + l\gamma) = k(\alpha, \beta) + l(\alpha, \gamma)$

另外，**Cauchy 不等式** 可用内积表示为：

$$
|(\alpha, \beta)| \le \sqrt{(\alpha, \alpha)} \sqrt{(\beta, \beta)}.
$$

> $$
 (a_1b_1 + a_2b_2 + \cdots + a_nb_n)^2 = (a_1^2 + a_2^2 + \cdots + a_n^2)(b_1^2 + b_2^2 + \cdots + b_n^2)
$$

### 向量的模、单位向量

向量的模 $\|\alpha\| = \sqrt{(\alpha, \alpha)} = \sqrt{a_1^2 + a_2^2 + \cdots + a_n^2}$。特殊地，若 $\|\alpha\| = 1$，此时称 $\alpha$ 为单位向量

### 向量模的性质

- 非负性：$\|\alpha\| \ge 0$，$\|\alpha\| = 0 \Leftrightarrow \alpha = 0$
- 齐次性：$\|k\alpha\| = |k| \|\alpha\|$
- 三角不等式：$\|\alpha + \beta\| \le \|\alpha\| + \|\beta\|$

> 如果 $\alpha$ 是非零向量，则 $\frac{1}{\|\alpha\|} \alpha$ 一定是单位向量，通过 $\alpha$ 求这个单位向量的过程叫将向量 $\alpha$ 单位化。

### 向量的夹角、正交向量

若 $\alpha, \beta$ 均不为 0，则它们的**夹角**定义为：

$$
\arccos \frac{(\alpha, \beta)}{\sqrt{(\alpha, \alpha)} \sqrt{(\beta, \beta)}}.
$$

> $(\alpha, \beta) = \|\alpha\| \|\beta\| \cos \theta,\quad \cos \theta = \dfrac{(\alpha, \beta)}{\|\alpha\| \|\beta\|}.$

若 $(\alpha, \beta) = 0$，则称 $\alpha$ 与 $\beta$ **正交**，记为 $\alpha \perp \beta$。

> 零向量与任意向量都正交，
> 两个非零向量正交当且仅当这两个向量的夹角为 $\dfrac{\pi}{2}$
