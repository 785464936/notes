# 洛必达法则

若

- $\displaystyle \lim_{ x \to x_{0} }f(x) = \displaystyle \lim_{ x \to x_{0} }g(x) = 0(\infty)$
- $f(x)$ $和$ $g(x)$ 在 $x_{0}$ 的某去心邻域内可导，且 $g^{\prime}(x) \neq 0.$
- $\displaystyle \lim_{ x \to x_{0} } \dfrac{f^{\prime}(x)}{g^{\prime}(x)}存在（或 \infty）$
则

$$\displaystyle \lim_{  x \to x_{0} } \dfrac{f(x)}{g(x)} = \lim_{ x \to x_{0} } \dfrac{f^{\prime}(x)}{g^{\prime}(x)}$$
